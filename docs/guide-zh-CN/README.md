Yii 2 JUI 扩展
=======================

这是 Yii 2 的 jQuery UI 扩展。它封装了 [jQuery UI widgets](http://jqueryui.com/) 作为 Yii 的 widgets，
并且使得可以在 Yii 应用程序中可以非常简单的使用 jQuery UI widgets。
